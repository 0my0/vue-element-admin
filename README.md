
## 启动

```bash

# enter the project directory
cd vue-element-admin

# install dependency
npm install

# develop
npm run dev
```

This will automatically open http://localhost:9527




## 发布

```bash
# build for test environment
npm run build:stage

# build for production environment
npm run build:prod
```

## 菜单
```bash
# 页面目录 
src/views

# 菜单配置
src/router/index.js
示例看 

```
